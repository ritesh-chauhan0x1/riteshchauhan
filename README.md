## 🚀 Installation

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Git (for cloning)
- Code editor (VS Code recommended)

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/ritesh-chauhan0x1/riteshchauhan.git
   cd riteshchauhan
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   # Or use a local server
   python -m http.server 8000
   # OR
   npx serve .
   ```

3. **View locally**
   - Navigate to `http://localhost:8000` (if using server)
   - Or open `index.html` directly in browser

## 🎨 Customization

### 🎯 Adding New Projects

1. **Update project data** in `js/projects.js`
2. **Add project images** to appropriate folder
3. **Update filter categories** if needed

### 🎨 Styling Changes
- **Colors**: Modify CSS variables in `css/base.css`
- **Animations**: Adjust timing in `css/animations.css`
- **Layout**: Update grid/flexbox in respective CSS files

### 📱 Adding New Sections
1. Create new HTML file
2. Add corresponding CSS file
3. Update navigation in `js/main.js`
